\## Description

This project is a Django-based web application that performs sentiment analysis using a pretrained NLP model.



\## Features

\- User input for text

\- Sentiment prediction (Positive/Negative/Neutral)

\- Web interface using Django



\## Technologies Used

\- Python

\- Django

\- Machine Learning (NLP)

\- Hugging Face Transformers



\## How to Run

1\. Clone the repository

2\. Install dependencies:

&#x20;  pip install -r requirements.txt

3\. Run server:

&#x20;  python manage.py runserver
